# 🌍 All India Villages API

## 📌 Project Overview
This project is a backend API built using Node.js and Express.js that provides access to village-level data across India. It allows users to fetch and manage information such as village name, district, and state.

---

## 🚀 Features
- Get all villages data
- Structured REST API
- Connected with MySQL database
- Clean and modular code architecture
- Easy to extend with more features like search and filters

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- MySQL
- Git & GitHub
